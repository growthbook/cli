## growthbook contextual-bandits delete-contextual-bandit-linked-feature

Unlink a feature from a Contextual Bandit

### Synopsis

Detaches a feature from this contextual bandit by removing it from the bandit's linked-feature list and cancelling any queued draft auto-publish. The feature's `contextual-bandit-ref` rule itself is left untouched.

```
growthbook contextual-bandits delete-contextual-bandit-linked-feature [flags]
```

### Examples

```
  growthbook contextual-bandits delete-contextual-bandit-linked-feature --id <id> --feature-id <id>
```

### Options

```
  -f, --feature-id string   The linked feature id [required]
  -h, --help                help for delete-contextual-bandit-linked-feature
  -i, --id string           The Contextual Bandit id [required]
```
