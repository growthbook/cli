## growthbook constant-revisions put-constant-revision-metadata

Update constant metadata in a draft revision

### Synopsis

Stages metadata changes (name, owner, description, project) on the draft. Pass `version: "new"` to auto-create a draft. The change is only applied to the live constant when the revision is merged.

```
growthbook constant-revisions put-constant-revision-metadata [flags]
```

### Examples

```
  growthbook constant-revisions put-constant-revision-metadata --key <key> --version-param <value>
```
